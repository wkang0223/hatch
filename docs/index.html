Built for idle Macs

🔒

Screen-lock detection

Uses macOS CGSession API to detect when your screen is locked. Your GPU only leases when you're away.

📊

IOKit GPU monitoring

Polls IOAccelStatistics2 every 30 seconds. Drops off the network the moment your GPU spikes above 5%.

🛡️

sandboxed execution

Jobs run as a restricted neuralmesh_worker OS user with macOS sandbox-exec profiles. No filesystem or network access beyond the job directory.

⚡

MLX & Metal native

Jobs run natively with Apple's MLX framework and PyTorch MPS backend. Zero virtualization overhead — full Metal GPU access.

💰

NMC credits

Earn off-chain NMC credits (Phase 1). Signed Ed25519 escrow receipts prove your earnings. On-chain Arbitrum settlement in Phase 3.

How it works

01

Install the agent

One-line installer. Creates the neuralmesh_worker isolation user, installs MLX & PyTorch MPS runtimes, and registers a launchd daemon.

02

Lock your screen

When your GPU util stays below 5% for 10 minutes with your screen locked, your Mac announces itself as available.

03

Jobs run automatically

The coordinator matches incoming jobs to your Mac. Jobs execute in an isolated sandbox. You earn NMC per GPU-hour.

04

Withdraw earnings

Cash out NMC credits to USDC on Arbitrum, or use them to submit your own compute jobs.
